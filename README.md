Projeto de aprendizado ebac, com Git e GitHub.

# Passo a passo (GitHub):
# 1- No GitHub - novo repositório, nomear e descrever, depois criar o novo repositório;
# 2- No vscode, com a pasta escolhida aberta, os comandos serão dados no terminal;
# 3- comando - git init: inicializa um repositório, transforma os arquivos da pasta em repositórios;
# 4- comando - git add .: para fazer o upload da pasta/repositório;
# * comando - git status: para acompanhar o status da criação, salvamento e upload do repositório;
# 5- comando - git commit -m "nome do 'salvamento' (procedimento feito)": 
# é o salvamento das ações feitas no código ou repositório;
# 6- comando - git branch -M main: cria uma branch principal chamada main;
# 7- comando - git checkout -b ebac: cria uma nova branch chamada ebac;
# 8- comando - git checkout main: volta para a branch main;
# 9- comando - git remote add origin + o endereço que está no GitHub:
# conecta a pasta local ao repositório criado no GitHub;
# 10- comando - git push -u origin main: envia o arquivo para o repositório criado;
# 11- comando - git push -u origin main (de novo...);
# Agora a branch 'main' estará no GitHub, na área de branchs.

# Quando um novo arquivo é criado dentro da pasta, no vscode, para adiciona-lo à branch existente:
# 1- comando - git add .: o novo arquivo é adicionado à branch;
# 2- comando - git commit -m "adicionando arquivo novo": para salvar a alteração ou atualização feita;
# 3- comando - git push -u origin main: para enviar a atualização para o repositório;

# *git add .: adiciona à branch sempre que uma atualização, novo código ou ação foi feita no projeto.
# *git commit -m "arquivo deletado": salva e descreve a ação realizada.
# *git push -u origin main: para enviar as informações à branch.
